Платформа для подготовки к ОГЭ по информатике с анализом ошибок, генерацией вариантов и персональными рекомендациями. 

      * Решение вариантов огэ и проверка их ответов
          -Количество правильных ответов и ошибок
          -Рекомендации по улучшению результатов

      * После решения варианта пользователь может запустить тренировку: Подтянуть слабые задания
          -Система автоматически формирует тренировку по неверным задачам

      * Генератор собственных вариантов

          -Пользователь может создать собственный вариант экзамена.
          -Настройки:
          Количество заданий
          Сложность
          Типы заданий
          Смешанный вариант
          
          Пример:
          Создать вариант:
          - 18 заданий
          - сложность: средняя
          - включить задания 17–18

      * В личном кабинете доступна подробная статистика:

            -средний балл
            -процент выполнения заданий
            -динамика прогресса

      * Для каждого задания доступна библиотека объяснений.

            -Видео берутся через API с образовательных платформ:
                    YouTube
                    VK Video
                    RuTube
